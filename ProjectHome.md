# Introduction #

A Unix command-line application for displaying Hijri date. This application can also converts Gregorian date to Hijri date.

## Usage ##

Prints current Hijri date:
```
hdate 
```

Convert Gregorian date to Hijri
```
hdate [day month year]
hdate 14 02 1990
```
